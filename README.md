# ScalatraHelloWorld
//build scala web app via `scalatra` framework


## Build & Run ##

```bash
cd ~ && cd ScalatraHelloWorld/app
sbt 
# inside sbt
> jetty:start
> browse
```
If `browse` doesn't launch your browser, manually open [http://localhost:8080/](http://localhost:8080/) in your browser.
